# Antigravity IDE Bundle

> Guia amigável e prático para configurar, usar e dar memória ao seu assistente de IA **Google Antigravity IDE** no Windows.

Este repositório contém material completo e gratuito para ajudar estudantes, profissionais de qualquer área, organizadores pessoais e iniciantes em tecnologia a instalar e usar o aplicativo **Google Antigravity IDE**. Você aprenderá a organizar estudos, TCC, tarefas pessoais e rotinas de trabalho diretamente nas pastas do seu computador, além de integrar o **Okam** para que a IA se lembre de todas as suas preferências.

### `/landing-page`
Código do site estático interativo com design Clean Tech Profissional moderno (inspirado no Google Gemini):

*   `index.html` — Landing page de página única com seções e abas.
*   `style.css` — Estilos customizados e design system roxo/índigo.
*   `script.js` — Lógica do FAQ, controle de abas, cópia de código e simulação interativa da IDE.
