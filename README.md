# Antigravity IDE Bundle

> Guia prático e open-source para configurar, usar e governar o **Google Antigravity IDE** no Windows.

Este repositório contém material completo e gratuito — licenciado sob MIT — para ensinar iniciantes, empresários, freelancers e programadores a instalar e utilizar a interface do aplicativo **Google Antigravity IDE**, além de integrar o **Okam** para governar regras e manter uma memória persistente e durável dos projetos.

## Estrutura do Repositório

### `/conteudo`
Guias completos no formato Markdown para leitura rápida direto na IDE ou no GitHub:

| Arquivo | Conteúdo |

| [`guia-introducao-antigravity-ide.md`](./conteudo/guia-introducao-antigravity-ide.md) | O que é o Antigravity IDE, como funciona a autonomia do agente e seus conceitos-chave. |
| [`tutorial-instalacao-windows.md`](./conteudo/tutorial-instalacao-windows.md) | Guia visual passo a passo para baixar o aplicativo, configurar a chave de API e primeiros passos. |
| [`guia-uso-basico-comandos.md`](./conteudo/guia-uso-basico-comandos.md) | Como usar o Chat, aprovar planos de implementação, gerenciar tarefas e revisar mudanças. |
| [`integracao-governanca-okam.md`](./conteudo/integracao-governanca-okam.md) | Dica especial: como usar o **Okam** complementarmente para governar as regras e manter a Wiki OKF. |

### `/landing-page`
Código do site estático interativo com design Clean Tech Profissional moderno (inspirado no Google Gemini):

*   `index.html` — Landing page de página única com seções e abas.
*   `style.css` — Estilos customizados e design system roxo/índigo.
*   `script.js` — Lógica do FAQ, controle de abas, cópia de código e simulação interativa da IDE.
