1. Guia de Introdução: Google Antigravity IDE

O **Google Antigravity IDE** é um assistente de inteligência artificial de desenvolvimento com capacidades autônomas (*agentic coding*) projetado pelo time do **Google DeepMind**. Ele não se limita a gerar respostas estáticas para você copiar: ele interage de forma proativa com os arquivos locais do seu repositório.

Por que ele é diferente de IAs tradicionais?

  • Capacidade de ação (Agency): Lê e escreve arquivos, examina diretórios com buscas avançadas e executa comandos locais.
  • Segurança por padrão: Não altera seu código sem aprovação. Toda ação complexa é apresentada em um plano.
  • Adaptação a regras locais: Carrega e respeita instruções específicas do seu projeto contidas no arquivo AGENTS.md.
💡 Conceito Chave

O Antigravity IDE opera em ciclos estruturados. Ele lê a tarefa, analisa os arquivos, monta um plano (implementation_plan.md) com as alterações exatas e aguarda você clicar em Aprovar para executar.

2. Instalação do Aplicativo no Windows

Configure seu workspace e ligue o cérebro do assistente seguindo os passos abaixo:

1

Download do Aplicativo

Baixe o executável do instalador do Antigravity IDE (ou instale como extensão diretamente do marketplace da sua IDE, buscando por "Google Antigravity IDE"). Conclua o assistente de instalação.

2

Obter API Key do Gemini

Acesse o Google AI Studio ↗ com sua conta Google, clique em Create API Key e copie o token gerado.

3

Configuração da Chave

Abra o Antigravity IDE, clique nas configurações (ícone de engrenagem) e cole a chave no campo Gemini API Key. Escolha o modelo recomendado (ex: Gemini 3.5 Flash) e salve.

4

Abrir o Workspace

Vá em File ➔ Open Folder e selecione a pasta do seu projeto. O chat lateral será inicializado automaticamente com acesso ao seu código.

3. Guia de Uso Básico no Chat e IDE

Veja como é o ciclo de interação com o agente na prática do dia a dia:

Como solicitar uma tarefa

Use o chat lateral para digitar em português normal o que deseja que o agente faça. Você pode arrastar arquivos para o chat ou usar @nome-do-arquivo para focar o contexto do agente.

Fluxo de Aprovação de Planos

Antes de modificar qualquer código, o agente abre uma janela propondo um Implementation Plan. Nele é mostrado:

  • Quais arquivos serão criados, modificados ou deletados.
  • Quais perguntas ele precisa que você responda antes de iniciar.

Você revisa e clica em Aprovar (Proceed) no botão da IDE ou digita alterações no próprio chat.

Visualização de Diffs

Ao editar arquivos, a IDE abrirá uma tela comparando o código original e as alterações propostas pelo agente. Isso permite revisar linha por linha antes de confirmar a escrita definitiva.

⚠️ Segurança

O agente sempre solicita aprovação gráfica no chat para executar comandos do terminal local (como rodar testes ou instalar dependências).

4. Governança e Memória Persistente com Okam

Por padrão, agentes de IA sofrem com "amnésia" entre conversas. Toda vez que você reinicia uma sessão de chat ou abre outro projeto, o agente esquece as diretrizes passadas e os aprendizados de bugs corrigidos.

💡 A Solução: Okam

O **Okam** é o framework de governança complementar perfeito para o Antigravity IDE. Ele atua criando regras persistentes e guardando o histórico das decisões técnicas do projeto.

Benefícios de usar o Okam com o Antigravity:

  • Constituição do Projeto (AGENTS.md): O Okam estrutura o arquivo de governança central que ensina as regras e preferências do projeto ao Antigravity IDE a cada nova sessão.
  • Memória de Longo Prazo (Wiki OKF): Grava lições aprendidas e documentações de arquitetura na Wiki local, garantindo que o agente consulte esse histórico em vez de repetir erros.
  • Segurança contra Vazamento de Segredos: Ativa Git Hooks automáticos que impedem o commit e push acidental de chaves privadas do Gemini ou segredos no GitHub.

Como inicializar o Okam no seu repositório:

Bash / PowerShell 
# 1. Instale o pacote oficial
pip install okam

# 2. Inicialize o setup de governança e hooks de commit
okam setup

Visite a documentação oficial para acessar guias e regras prontas em: okam-os.vercel.app ↗
