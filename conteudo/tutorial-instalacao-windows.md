# Tutorial de Instalação no Windows: Google Antigravity IDE

Instalar novos programas pode parecer complicado à primeira vista, mas prometo: você **não vai precisar de nenhum conhecimento técnico aqui**. Este guia mostra exatamente o que clicar e digitar, sem pular etapas e sem termos difíceis.

Siga os passos na ordem abaixo e, em cerca de 15 minutos, você terá um assistente de inteligência artificial funcionando no seu computador.

---

## 🔽 Passo 1: Baixar e instalar o Antigravity IDE

O Antigravity IDE é o programa principal — é ele que você vai abrir toda vez que quiser conversar com a IA.

1. Abra o seu navegador de internet (pode ser o Chrome, Edge ou qualquer outro).
2. Acesse a página oficial: [antigravity.google/download#antigravity-ide](https://antigravity.google/download#antigravity-ide)
3. Você verá um botão para baixar o arquivo de instalação para Windows. O nome do arquivo vai terminar em `.exe` (por exemplo: `antigravity-ide-setup.exe`). Clique nele para iniciar o download.
4. Quando o download terminar, abra a pasta **Downloads** do seu computador (geralmente fica no Explorador de Arquivos, aquela pastinha amarela na barra de tarefas) e dê **dois cliques** no arquivo que você acabou de baixar.
5. Uma janela de instalação vai aparecer na tela. Clique em **Próximo** (ou **Next**) em cada tela que aparecer, sem alterar nada, até chegar no botão **Instalar** (ou **Install**). Clique nele e aguarde a barra de progresso terminar.
6. Quando aparecer a tela final com **Concluir** (ou **Finish**), clique nele. O programa está instalado!

> 💡 **Se aparecer uma janela de segurança do Windows** perguntando *"Deseja permitir que este aplicativo faça alterações no dispositivo?"*, clique em **Sim**. Isso é completamente normal — é o Windows pedindo confirmação para instalar o programa.

---

## 🔑 Passo 2: Fazer login com sua conta Google

Para usar a inteligência artificial, você precisa se identificar com a sua conta Google — a mesma que você já usa no Gmail, YouTube ou Google Drive.

1. Abra o **Antigravity IDE**: clique no botão do Windows (o ícone no canto inferior esquerdo da tela), digite **Antigravity** e clique no ícone do programa que aparecer.
2. Você verá a tela inicial do programa. Clique no botão **Entrar** (ou **Sign in**).
3. O seu navegador de internet vai abrir automaticamente uma página do Google pedindo que você escolha uma conta.
4. Clique na sua conta Google e confirme o acesso quando solicitado.
5. Volte para o programa — ele já estará conectado. A tela vai mudar e mostrar a área principal do Antigravity IDE.

> 💡 **Você não precisa criar nenhuma conta nova.** Use a sua conta Google que já existe.

---

## 📁 Passo 3: Abrir a sua pasta de trabalho

O Antigravity IDE trabalha sempre dentro de uma pasta do seu computador. É nessa pasta que ele vai criar e organizar os seus arquivos.

1. Primeiro, **crie uma pasta nova** no seu computador para começar. Aqui vão algumas sugestões de nome: `MeuTCC`, `MinhasFinancas`, `Estudos`, `TrabalhoDoJoao`.

   Para criar a pasta: abra o **Explorador de Arquivos** (a pastinha amarela na barra de tarefas), navegue até **Documentos** e clique com o **botão direito do mouse** em um espaço vazio → escolha **Novo** → **Pasta**. Digite o nome e pressione **Enter**.

2. De volta ao Antigravity IDE, clique no menu **File** (Arquivo) na parte superior da janela, depois clique em **Open Folder** (Abrir Pasta).
3. Uma janela vai se abrir para você escolher uma pasta. Encontre a pasta que você acabou de criar, clique nela uma vez para selecioná-la e depois clique em **Selecionar Pasta** (ou **Select Folder**).
4. O assistente vai ler rapidamente o conteúdo da pasta. Em seguida, o painel de chat vai aparecer na lateral direita da tela.

**A IA já está pronta para conversar!** Mas antes de começar a usar de verdade, faça os próximos dois passos para instalar os programas auxiliares — e o melhor: a própria IA fará isso por você!

---

## 🔧 Passo 4: Pedir para a IA instalar o Git

O **Git** é um programa que fica funcionando nos bastidores do seu computador. Você nunca vai precisar abri-lo diretamente — ele é usado silenciosamente pelo Antigravity IDE para guardar versões do seu trabalho, como uma máquina do tempo para seus arquivos.

**Você não precisa instalar o Git manualmente.** Basta pedir para a IA fazer isso por você!

1. Olhe para o painel de chat na lateral direita do Antigravity IDE. Você vai ver uma caixa de texto na parte de baixo onde pode digitar mensagens.
2. **Copie o texto abaixo** (selecione tudo, clique com o botão direito → Copiar):

```text
Por favor, verifique se o Git está instalado no meu computador Windows. Se não estiver, baixe e instale o Git para Windows de forma automática e me avise quando terminar.
```

3. **Cole o texto** na caixa de chat (clique dentro da caixa e pressione as teclas **Ctrl + V** ao mesmo tempo) e depois pressione **Enter** no teclado.
4. O assistente vai verificar se o Git já existe no seu computador e, se não existir, vai mostrar um card na tela com o título **Plano de Implementação** descrevendo o que vai fazer.
5. Clique no botão **Aprovar** (ou **Proceed**) para dar permissão.
6. Aguarde. Em alguns minutos, o chat vai exibir uma mensagem avisando que o Git foi instalado com sucesso.

> 💡 **Por que preciso do Git?** Pense nele como um "controle de versões automático". Se você editar um arquivo e se arrepender da mudança, o Git pode ajudar a recuperar a versão anterior. A IA usa isso para proteger o seu trabalho.

---

## ⚙️ Passo 5: Pedir para a IA instalar o Node.js

O **Node.js** é outro programa que trabalha nos bastidores. Ele permite que o Antigravity IDE crie coisas mais avançadas: sites interativos, planilhas dinâmicas e automações inteligentes.

Assim como o Git, você não precisa instalar manualmente — basta pedir para a IA!

1. Na caixa de chat do Antigravity IDE, **copie e cole** o texto abaixo e pressione **Enter**:

```text
Por favor, verifique se o Node.js está instalado no meu computador Windows. Se não estiver, baixe e instale a versão LTS mais recente do Node.js de forma automática e me avise quando terminar.
```

2. O assistente vai verificar e, se necessário, mostrar um card pedindo sua autorização.
3. Clique em **Aprovar** para confirmar.
4. Aguarde o aviso de conclusão no chat.

> 💡 **"LTS" significa *Long Term Support***, ou seja, a versão mais estável e recomendada para uso geral. É sempre a escolha certa para iniciantes.
